# Reversi Game

An interactive browser-based implementation of Reversi, built with
HTML, CSS, and JavaScript.

## Live Demo

The playable demo is hosted using GitHub Pages.

## Features

- Complete 8 × 8 Reversi board
- Eight-direction move validation
- Automatic piece flipping
- Valid-move highlighting
- Turn management
- Automatic pass handling
- Score tracking
- Win and tie detection
- Responsive browser interface

## Technologies

- HTML5
- CSS3
- JavaScript
- DOM manipulation
- Git and GitHub Pages

## Project Background

This browser demo is an independent implementation of the Reversi
game rules.

A separate C++20 implementation uses object-oriented programming,
polymorphism, and the Observer design pattern. That source code is
maintained in a restricted private repository to comply with academic
course policies.

Private source access is available to potential employers upon request.
